Lesson 24: Responsive Design

Make your web pages look great on every screen — phone, tablet, or desktop.

Concept

What Is Responsive Design?

A responsive web page automatically adjusts its layout, font sizes, images, and spacing to fit the screen it is displayed on. There is only one HTML file — CSS does all the heavy lifting.

The Three Pillars 1. Flexible layouts (%, fr, flexbox, grid)
2. Fluid images (max-width: 100%)
3. Media queries (@media) for breakpoints
HTML

The Viewport Meta Tag

This tag lives in <head> and is required for responsive design to work on mobile browsers. Without it, phones zoom out and ignore your media queries. 

<!-- Required in every responsive page -->
<meta name="viewport"
      content="width=device-width, initial-scale=1.0" />

<!-- width=device-width  → match the real screen width
     initial-scale=1.0  → no automatic zoom on load -->
CSS Pattern

Mobile-First CSS Approach

Write your base styles for mobile first — no media query needed. Then use min-width media queries to add styles for larger screens. 

/* BASE (mobile, < 600px) — no media query */
.container {
  padding: 0 16px;
  font-size: 1rem;
}

/* TABLET — 600px and above */
@media (min-width: 600px) {
  .container {
    padding: 0 32px;
    font-size: 1.05rem;
  }
}

/* DESKTOP — 900px and above */
@media (min-width: 900px) {
  .container {
    max-width: 1100px;
    margin: 0 auto;
    font-size: 1.1rem;
  }
}
min-width vs max-width Mobile-first uses min-width (styles apply from this width upward). Desktop-first uses max-width (styles apply up to this width). Mobile-first is the industry standard today.
Grid Layout

3-Column Grid → 1-Column on Mobile

Resize this window to see the cards reflow. On mobile they stack in one column; on tablet they become two columns; on desktop, three columns. 

/* MOBILE: single column (default) */
.feature-grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

/* TABLET: 2 columns */
@media (min-width: 600px) {
  .feature-grid {
    grid-template-columns: 1fr 1fr;
  }
}

/* DESKTOP: 3 columns */
@media (min-width: 900px) {
  .feature-grid {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
📱

Mobile First

Design for the smallest screen, then enhance upward. Content always works on every device.

🔍

Media Queries

Apply different CSS rules at specific viewport widths using the @media at-rule.

⚙

Flexible Units

Use %, em, rem, vw, and fr instead of fixed pixels so elements scale naturally.

📸

Fluid Images

Images with max-width: 100% shrink on small screens and never overflow their container.

📋

Grid & Flexbox

Modern CSS layout tools that create flexible, multi-column designs without floats.

🌐

Viewport Meta

The meta viewport tag tells mobile browsers to match the real screen width.

Images

Fluid Images

The single most important CSS rule for images is max-width: 100%. It lets images shrink on narrow screens while never growing beyond their natural size. 

/* Makes all images responsive */
img {
  max-width: 100%;  /* shrink to fit container */
  height: auto;     /* keep aspect ratio */
  display: block;   /* remove inline spacing */
}
Navigation

Collapsing Navigation Bar

Scroll to the top of this page and resize the window. Below 600px a hamburger button appears. Click it to reveal the navigation links stacked vertically. Above 600px the links are always visible in a horizontal row. 

/* MOBILE: hide links; show hamburger */
.nav-links { display: none; }
.nav-links.open { display: flex; flex-direction: column; }
.nav-toggle { display: flex; }

/* TABLET+: show links; hide hamburger */
@media (min-width: 600px) {
  .nav-links  { display: flex !important; flex-direction: row; }
  .nav-toggle { display: none; }
}
Auto-Fill Grid

Student Card Grid (Reflows Automatically)

Using repeat(auto-fill, minmax(220px, 1fr)) the browser decides how many cards fit per row — no explicit breakpoints needed. Resize the window to watch the cards reflow. 

/* No breakpoints needed — auto-fill handles it */
.student-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(220px, 1fr));
  gap: 20px;
}
/* Each card is at least 220px wide.
   As many columns as can fit are created.
   Cards grow equally to fill remaining space. */

Reference

Quick Reference

/* 1. Always include the viewport meta tag */
<meta name="viewport" content="width=device-width, initial-scale=1.0">

/* 2. Fluid images */
img { max-width: 100%; height: auto; }

/* 3. Mobile-first media queries */
/* base = mobile, no query needed */
@media (min-width: 600px) { /* tablet  */ }
@media (min-width: 900px) { /* desktop */ }

/* 4. Flexible grid */
.grid { display: grid; grid-template-columns: 1fr; }
@media (min-width: 600px) { .grid { grid-template-columns: 1fr 1fr; } }
@media (min-width: 900px) { .grid { grid-template-columns: 1fr 1fr 1fr; } }

/* 5. Auto-reflow grid (no breakpoints) */
.auto-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(220px, 1fr));
}
