Section: Paragraphs

This is a paragraph. Paragraphs are blocks of text used to write sentences and descriptions. The browser automatically adds space above and below each paragraph.

This is a second paragraph. Notice that even though there is no blank line in the HTML code between the two paragraphs, the browser still separates them visually. HTML ignores extra whitespace — the tags control the layout.

Section: Bold and Italic Text

You can make text bold using the strong tag to emphasize important words.

You can make text italic using the em tag to add emphasis or highlight terms.

You can combine them: this text is both bold and italic.

Section: Ordered List (Numbered Steps)

Use an ordered list when the ORDER of items matters (like steps in a process):

  1. Open a text editor (like VS Code or Notepad)
  2. Write your HTML code
  3. Save the file with a .html extension
  4. Open the file in a web browser
  5. See your webpage displayed!

Section: Div and Span (Grouping Elements)

Two very commonly used tags are div and span. They have no visual appearance on their own — they are used to GROUP content so you can style or organize it.

This entire blue box is wrapped in a single div. The background color and border are applied to the div itself, which groups everything inside it.

Divs are one of the most used elements in web development for layout and organization.

In a paragraph, you can highlight specific words using a span without breaking the flow of the sentence.

Another example: the word important is wrapped in a span so it can be styled red and bold independently of the rest of the sentence.

What You Learned in Lesson 21

Great work making it through your first HTML lesson! Here is a summary of every tag you encountered:

Tag What It Does
<!DOCTYPE html>Declares this is an HTML5 document
<html>The root element that wraps the entire page
<head>Contains page metadata (not visible to users)
<title>Sets the text shown in the browser tab
<body>Contains all visible page content
<h1> – <h6>Headings from largest (h1) to smallest (h6)
<p>A paragraph of text
<strong>Bold text (with semantic importance)
<em>Italic text (with semantic emphasis)
<a>A hyperlink to another page or resource
<img>Embeds an image (self-closing, no closing tag)
<ul> / <li>Unordered (bullet point) list and list items
<ol> / <li>Ordered (numbered) list and list items
<table>, <tr>, <th>, <td>Table, row, header cell, and data cell
<div>A block-level container for grouping elements
<span>An inline container for styling part of text
<hr>A horizontal dividing line
