CSS Styling

Learn how to make your HTML look great using Cascading Style Sheets

1. Selectors

CSS selectors tell the browser which elements to style. The three most common types are:

  • p { } — Element selector: targets every <p> tag
  • .card { } — Class selector: targets anything with class="card"
  • #header { } — ID selector: targets the one element with id="header"

A class is like a label you can apply to many elements. An ID is a unique name for one element only.

2. Colors

You can set color on the background, text, and border of any element. CSS accepts color names, hex codes, rgb(), and rgba() values.

3. The Box Model

Every element is a box with four layers. From outside to inside: margin → border → padding → content.

Margin (orange dashed) — space outside the border
Border (blue solid) — the visible edge
Padding (green dashed) — space inside the border
Content — the actual text or image here
